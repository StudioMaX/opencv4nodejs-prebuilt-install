### Simple installation Opencv 4.1.1 for node with pre-compiled bindings

Cross-platform!

#### Supports

- Windows, Linux , MacOS
- node 12,13,14,15,16,17,18,19,20
- arh x64

## How to install

``` bash
npm i opencv4nodejs-prebuilt-install
```
